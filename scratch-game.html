Overview

This project is a Scratch-based game built from scratch as an interactive logic project. It shows creativity, control flow thinking, user interaction design, and the ability to turn an idea into a playable experience.

What this project demonstrates

  • Interactive thinking
  • Event-based logic
  • Basic gameplay system design
  • Creativity in building playable experiences
  • Structuring user interaction

Things Learned

  • How game logic depends on states and interactions
  • How user feedback changes the feel of a project
  • How even beginner-friendly tools can still teach real logic and design thinking
